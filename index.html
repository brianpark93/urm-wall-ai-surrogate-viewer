What this is (and isn't)

🧠
Live inference, not a lookup table

Both networks were trained once (on all 185 IDA runs) and exported as plain weight matrices. The forward pass for whatever PGA you type runs directly in this page's JavaScript — every value, including ones never simulated, is a genuine model prediction.

📏
Training range: 0.01–0.66g

PGA above 0.66g is extrapolation beyond every simulated run (flagged above). Interpolated values (e.g. 0.153g) were never simulated either, but fall inside the trained range.

⚖️
PINN is not "more accurate" here

Leave-one-out testing showed the physics-informed model is not more accurate than the plain one on this dataset (which has documented non-monotonic damage transitions) — it's shown side-by-side purely to compare what the monotonicity constraint changes qualitatively, especially on extrapolation.
